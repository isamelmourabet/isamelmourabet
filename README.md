## 🚀 About Me

I'm a software and hardware developer based in Madrid, currently finishing my degree in Computer Engineering at URJC. I am passionate about building *end-to-end* products, from low-level architecture and board design to user interfaces in mobile applications.

* 💻 **My focus:** Writing clean code, designing scalable architectures, and creating seamless user experiences.
* 🔧 **What I build:** From native applications in the Apple ecosystem (iOS/visionOS) to embedded systems, IoT, and platforms with Web3 integration.
* 🚀 **Currently:** Building my own startup, **Owllink**, and developing hardware and software for the health sector.
